# Focusly — Pomodoro Task Manager

Focusly is a clean and minimal productivity web app that combines a **Pomodoro timer**, **task manager**, and **focus environment controls** to help users stay productive and organized.

It is designed with a distraction‑free interface and soft visual aesthetics to make long study or work sessions comfortable and effective.

---

## ✨ Features

### ⏱ Pomodoro Timer

* 25‑minute default focus session
* Short Break mode
* Long Break mode
* Circular visual timer
* Play / Pause controls
* Session reset and skip options

### 📊 Productivity Stats

* Pomodoros completed today
* Total focus time tracked
* Task progress counters:

  * Total tasks
  * Completed tasks
  * Pending tasks

### 📝 Task Manager

* Add tasks quickly
* Set task category (e.g., Work)
* Set priority level (Low / Medium / High)
* Estimated pomodoros required
* Task filters:

  * All
  * Pending
  * Done

### 🎧 Focus Environment

Built‑in ambient sound options to improve concentration:

* Rain
* Café
* Forest
* Waves
* Off option

### 📅 Weekly Focus Overview

* Visual tracker for weekly productivity
* Day‑wise focus indicators (Mon–Sun)

### 💬 Motivation Panel

* Rotating motivational quotes
* Clean card layout for visual comfort
